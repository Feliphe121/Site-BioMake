Curiosidades sobre o Projeto

  • O projeto utilizou tecnologias como Raspberry Pi para automação e controle.
  • A impressão 3D foi uma das ferramentas principais para criar protótipos rápidos.
  • O projeto envolveu o uso de sensores para otimizar o desempenho de foguetes de garrafa PET.

Impacto e Contribuições

Este projeto teve um impacto significativo na educação e na divulgação de ciência, permitindo a estudantes e entusiastas entenderem melhor as aplicações de tecnologias como Internet das Coisas (IoT) e impressão 3D de forma prática e interativa.
